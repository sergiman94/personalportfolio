Un desarrollador web de back-end es responsable de la aplicación web del lado del servidor con la lógica e integración del trabajo que realizan los desarrolladores web front-end. los desarrolladores back-end suelen escribir servicios web y API's que proporcionan recursos de informacion que son utilizados por los desarrolladores front-end como tambien por otros desarrolladores enfocados en demas dispositivos.                        


De hecho, las herramientas que uso para el desarrollo de back-end son Express framework y Node.js. para gestion de bases de datos uso PostgreSQL, MySQL y SQLDeveloper. Si Se necesitan otros marcos o entornos de desarrollo estoy en disposición de utilizarlos y aprender de ellos.

Desarrollo web front-end, también conocido como desarrollo del lado del cliente es la práctica de producir HTML, CSS y JavaScript para un sitio web o aplicación web para que un usuario pueda verlos e interactuar con ellos directamente.


El objetivo de diseñar una plataforma web es asegurar que cuando los usuarios lo usen, vean la información en un formato que es fácil de leer y relevante. Esto es un trabajo de analisis y diseño profundo por el hecho de que los usuarios ahora usan Una gran variedad de dispositivos con diferentes tamaños de pantalla y resoluciones, obligando al diseñador a tener en cuenta estos aspectos cuando se encuentra diseñando y desarrollando el sitio. Deben asegurarse de que todos los componentes se encuentren distribuidos correctamente en diferentes navegadores (cross-browser), diferentes sistemas operativos (multiplataforma) y diferentes dispositivos (dispositivo cruzado), que requiere una planificación cuidadosa del lado del desarrollador.
